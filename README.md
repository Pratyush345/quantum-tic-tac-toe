# Quantum Tic-Tac-Toe

A quantum-inspired version of Tic-Tac-Toe that incorporates probabilistic state evolution and quantum computing concepts. Developed as part of IEEE Quantum Computing activities to demonstrate how ideas such as superposition, probability amplitudes, and state collapse can be visualized through gameplay.

## Overview

Unlike classical Tic-Tac-Toe, each move influences the probability distribution of future game states. The game continuously evaluates and updates the likelihood of different outcomes after every move, allowing players to observe how the board evolves under quantum-inspired rules.

The project serves as an educational tool for understanding quantum computing concepts through an interactive game environment.

## Features

- Quantum-inspired Tic-Tac-Toe gameplay
- Probability calculation after every move
- Dynamic prediction of game outcomes
- Real-time state updates
- Interactive GUI using Tkinter
- Educational visualization of quantum concepts

## Quantum Concepts Demonstrated

- Superposition
- Probability distributions
- State evolution
- State collapse and outcome prediction
- Quantum-inspired decision making

## How It Works

1. Players make moves on the board.
2. The game updates the underlying probabilistic state.
3. Outcome probabilities are recalculated after every move.
4. The interface displays evolving predictions and game-state information.
5. The game continues until a final outcome is reached.
